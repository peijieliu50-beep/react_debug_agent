# 说话人识别实验指导书（课程实验）

## 1. 实验目标
基于深度学习实现说话人识别（Speaker Recognition）：给定一段语音，判断说话人身份。
属于音频分类任务，可用 CNN / TDNN / Transformer 等结构。

## 2. 数据与特征
- 原始音频需提取声学特征，常用 MFCC（梅尔频率倒谱系数）或 Mel 频谱图。
- 特征维度通常为 (时间帧数, 特征维)，输入网络前注意 batch 维与通道维。
- 标签为说话人 ID（整数类别），分类用 LongTensor。

## 3. 典型网络结构
- 输入：Mel 频谱图，形如 (batch, 1, n_mels, n_frames)。
- 卷积层提取局部特征 → 池化 → 全连接 → softmax 分类。
- 关键：卷积输出展平送入 Linear 时，in_features 要与展平维度一致，否则报维度不匹配。

## 4. 常见实验问题
- 输入特征维度与网络第一层不匹配：核对 n_mels 与 Conv 的 in_channels。
- 类别数与最后一层输出维度不符：Linear 最后输出应等于说话人数量。
- 音频长度不一：需统一裁剪/填充到固定长度，或使用自适应池化。
- 训练集小易过拟合：加 Dropout、数据增强（加噪、变速）。

## 5. 评估指标
- 准确率（Accuracy）：分类正确比例。
- 等错误率（EER）：说话人确认任务常用。
- 混淆矩阵：分析易混淆的说话人。
